edgeX Agent — Navigation Blueprint v3

3 modules: Trade / AI Agent / News. Always 3 buttons on L1.

Architecture: 3 Modules

  • L1 Dashboard — always 3 buttons. Top = Trade (or Connect edgeX). Middle = AI Agent (or Activate AI). Bottom = News.
  • L2 Hubs — Trade hub (account + positions + sub-actions), AI Agent hub (personality + provider + memory), News (sources + frequency).
  • L3 Sub-screens — Close Position, Orders, History, News Freq, Add Source, Memory, Clear Memory. Back → parent L2.
  • Push Layer — News alerts, AI chat, trade plan/result. New messages, edit-in-place for confirm→result.
