Navigation Rules

  • L1 Dashboard — /start sends new message. Always 3 buttons. All navigation edits this message in-place.
  • L2 Hubs — Each module's main screen. Back → Dashboard.
  • L3 Sub-screens — Deeper actions within a module. Back → parent L2 Hub.
  • Push Layer — Bot-initiated messages (news alerts, AI chat, trade results). Separate from dashboard. Confirm→Execute→Result edits the same push message.
  • Arrows → show navigation direction. All within a module. No cross-module jumps except back to Dashboard.
