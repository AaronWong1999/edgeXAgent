# edgeX Agent

AI-powered trading agent for [edgeX Exchange](https://edgex.exchange), accessible through Telegram.

Trade 60+ perpetual contracts (crypto, stocks, commodities) by chatting in any language. The bot handles market data, AI analysis, trade execution, and portfolio management — all inside Telegram.

**Website:** [edgexagent.com](https://aaronwong1999.github.io/edgeXAgent/)

**Bot:** [@edgeXAgentBot](https://t.me/edgeXAgentBot)

## Highlights

- Natural language trading in 6+ languages
- 7 AI personalities (Degen, Sensei, Professor, Wolf, Moe, ...)
- Per-user memory system (remembers your preferences and past conversations)
- Real-time portfolio injection (AI never hallucinates your positions)
- Button-first UX inspired by PepeBoost
- Bring your own AI key (OpenAI, DeepSeek, Claude, Gemini, Groq)
- Admin dashboard for feedback and user conversation review
